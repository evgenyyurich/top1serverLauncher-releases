# top1server Launcher — релизы

Публичный репозиторий с готовыми сборками лаунчера **top1server**.

**Скачать:** вкладка [Releases](../../releases/latest) → файл `Top1ServerLauncher.exe`.

Исходный код лаунчера — в приватном репозитории. Здесь публикуются только бинарники,
которые лаунчер использует для авто-обновления.
